LAMP Stack Arch Linux

04-02-2021 (updated )   #LAMPSTACK   baca 2 menit

Hallo, apa kabar semua ? Pasti baik-baik saja kan 😄

By the way, LAMP (Linux Apache MariaDB PHP) merupakan sebuah paket perangkat lunak bebas yang digunakan untuk menjalankan sebuah aplikasi secara lengkap. Biasanya digunakan untuk menjalankan aplikasi berbasis web.

1. Install Apache

Apache merupakan server web yang dapat dijalankan di banyak sistem operasi. Apache sendiri dikembangkan oleh komunitas terbuka yang terdiri dari pengembang-pengembang di bawah naungan Apache Software Foundation.

Lokasi direktori apache default terdapat di /srv/http dan untuk lokasi konfigurasi filenya ada di /etc/httpd/conf.

Jalankan terminal dan ketikan perintah di bawah ini supaya system tetap up-to-date:

1

 $ sudo pacman -Syy && pacman -Syu

Lalu, install apache

1

 $ sudo pacman -S apache

Start apache service

1

 $ sudo systemctl start httpd.service

Cek apache service

1

 $ systemctl status httpd.service

Lalu, cari dan uncomment (hilangkan tanda #) kode di bawah ini pada /etc/httpd/conf/httpd.conf

1

 #LoadModule rewrite_module modules/mod_rewrite.so

2. Install MariaDB

MariaDB di Arch Linux merupakan implementasi MySQL. Jalankan perintah dibawah ini untuk install mariadb.

1

 $ sudo pacman -S mariadb

Jalankan perintah dibawah ini sebelum memulai mariadb.service

1
2


 $ sudo mariadb-install-db --user=mysql --basedir=/usr --datadir=/var/lib/mysql

Start mariadb service

1

 $ sudo systemctl start mariadb.service

Cek mariadb service

1

 $ systemctl status mariadb.service

3. Install PHP

PHP (Hypertext Preprocessor) merupakan bahasa yang cocok untuk Web development dan bisa disisipkan dalam HTML.

Jalankan perintah dibawah ini untuk install php dan ekstensi yang biasanya digunakan.

1

 $ sudo pacman -S php php-apache php-cgi php-gd php-imap

Konfigurasi php terdapat /etc/php/php.ini

1

 $ sudo nano /etc/php/php.ini

Cari dan ubah konfigurasinya (hilangkan tanda ; ).

Time zone

1

 date.timezone = Asia/Jakarta

Display error

1

 display_errors = On

PHP GD ekstension

1

 extension=gd

PHP Imap ekstension

1

 extension=imap

MySQL/MariaDB ekstension

1
2

extension=pdo_mysql
extension=mysqli

Ubah konfigurasi di apace /etc/httpd/conf/httpd.conf.

berikan tanda # pada kode dibawah ini

1

 #LoadModule mpm_event_module modules/mod_mpm_event.so

hapus tanda # pada kode dibawah ini

1

 LoadModule mpm_prefork_module modules/mod_mpm_prefork.so

Lalu tambahkan LoadModule di akhir baris kode dibawah ini pada konfiguarasi /etc/httpd/conf/httpd.conf.

PHP LoadModule

1
2
3

LoadModule php_module modules/libphp.so
AddHandler php-script .php
Include conf/extra/php_module.conf

Terakhir, restart apache.

1

 $ systemctl restart httpd.service

Membuat test php file. untuk cek php bekerja pada apache atau tidak.

1

 $ sudo nano /srv/http/info.php

Tambahkan kode dibawah ini

1

 <?php phpinfo(); ?>

Lalu buka web browser dan kunjungi http://localhost/info.php atau http://127.0.0.1/info.php.

4. Install PHPmyAdmin

PHPmyAdmin merupakan tools untuk manage database berbasis web.

Jalankan kode dibawah ini untuk install phpmyadmin

1

 $ sudo pacman -S phpmyadmin

Buat file phpmyadmin.conf

1

 $ sudo nano /etc/httpd/conf/extra/phpmyadmin.conf

Lalu, copy paste kode dibawah ini

1
2
3
4
5
6
7

Alias /phpmyadmin "/usr/share/webapps/phpMyAdmin"
<Directory "/usr/share/webapps/phpMyAdmin">
    DirectoryIndex index.php
    AllowOverride All
    Options FollowSymlinks
    Require all granted
</Directory>

Buka pada konfigurasi apache

1

 $ sudo nano /etc/httpd/conf/httpd.conf

Tambahkan pada konfigurasi apache

1
2

# phpMyAdmin configuration
Include conf/extra/phpmyadmin.conf

Terakhir, restart apache.

1

 $ systemctl restart httpd.service

Lalu buka web browser dan kunjungi http://localhost/phpmyadmin atau http://127.0.0.1/phpmyadmin.

Jika ada yang perlu ditanyakan. Tanyakan saja di komentar. 😃
